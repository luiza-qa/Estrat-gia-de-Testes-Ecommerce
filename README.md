# 🛡️ Estratégia de Testes: E-commerce de Alta Performance
**Responsável:** Luíza Silva — QA Engineer (Manual & API)

---

### 📋 Visão Geral do Projeto
Este projeto documenta o ciclo completo de Garantia de Qualidade (QA) para uma plataforma de E-commerce. O foco principal é validar a jornada do usuário, desde a busca de produtos até a finalização do checkout, garantindo um sistema livre de bugs críticos e uma experiência fluida.

### 🎯 Artefatos de QA Produzidos
Neste repositório, você encontrará a documentação técnica real utilizada no projeto:

1.  **Plano de Testes:** Definição de escopo, riscos e cronograma.
2.  **Suíte de Casos de Teste:** Passo a passo detalhado das validações funcionais.
3.  **Relatórios de Bugs (Bug Reports):** Documentação de falhas encontradas com evidências e severidade.
4.  **Testes de Regressão:** Garantia de que novas funcionalidades não quebraram o que já estava pronto.
